uScribe is a personal AI-powered writing assistant built with Electron, React, and TypeScript.

Key features:
- Rich note editor for drafting and organizing notes
- AI autocomplete while you type
- Inline text revision with accept/reject preview
- Global shortcuts for revising selected text in other apps
- Configurable AI providers (OpenAI, Azure OpenAI, Claude, Ollama, LiteLLM)
